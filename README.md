# Codveda Frontend Internship Tasks

This repository contains the frontend tasks completed as part of my internship at Codveda Technology.

The work is organized level-wise to show a clear progression from basic concepts to more advanced frontend skills.

## Tasks Overview

- **Level 1:** Interactive form built using HTML, CSS, and JavaScript with a focus on user experience.
- **Level 2:** REST API integration project that fetches and displays dynamic data using a public API.
- **Level 3:** Advanced UI animations implemented using GSAP, focusing on smooth interactions and performance.

Each level has its own folder with source code and a brief README explaining the task and learning outcomes.

**Tech Stack:** HTML, CSS, JavaScript, GSAP
